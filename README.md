# GraduationAlbum

## 对于展示信息的定义


```javascript
		yourBooksInfo : [{
			_id:'',
			yourDataId:'',
			photoUrl:'',
			leaveMsg:'',
			bookTemplate:''
		}]
	
```
存放的为数组形式，以保证在未来可以让用户定义多张文本

yourDataId存放的是个人信息的_id
